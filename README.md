# 🍹 BarDrinks — каталог напитков на Django

Учебный Django‑проект: каталог барных напитков с рейтингами, комментариями, ценами и случайным выбором напитка.

Выполнили: Сальцев Даниил, Майоров Дмитрий, Пантелеев Адриан, Козловский Иван

---

## 🚀 Функционал

- 📋 Список напитков
- 📄 Страница напитка
- ❤️ Лайки
- ⭐ Рейтинг (1–5, среднее значение)
- 💬 Комментарии
- 🎲 Случайный напиток («Мне повезёт»)
- 💰 Цена
- 🥃 Объём (мл)
- 🔥 Крепость (% алкоголя)
- 🗂 Категории
- 🔍 Поиск
- 🖼 Главная страница с картинкой
- 🛠 Админка Django

---

## 🧱 Стек технологий

- Python 3.12
- Django 6.0
- HTML + CSS
- Django Templates
